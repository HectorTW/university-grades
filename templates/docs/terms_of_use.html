Пользовательское соглашение (Публичная оферта)

Настоящее соглашение определяет условия использования сервиса «СтартТруд» всеми категориями пользователей: соискателями (студентами), работодателями и экспертами (преподавателями).

1. Предмет соглашения

Сервис предоставляет пользователям доступ к функционалу по ведению учебных профилей, выставлению оценок, отображению успеваемости и взаимодействию между студентами и работодателями.

2. Регистрация и учётная запись
  • для доступа к сервису пользователь создаёт учётную запись (логин и пароль);
  • пользователь обязуется предоставлять достоверные и актуальные данные;
  • ответственность за сохранность логина и пароля несёт сам пользователь.
3. Обязанности пользователя
  • не нарушать права других пользователей и третьих лиц;
  • не использовать сервис для рассылки спама, оскорблений, противоправного контента;
  • соблюдать академическую и деловую этику при использовании сервиса.
4. Ограничение ответственности
  • сервис не гарантирует достижение пользователем конкретных результатов (например, трудоустройства);
  • администрация не несёт ответственности за решения работодателей, принятые на основе информации из сервиса;
  • сервис может временно ограничивать доступ при проведении технических работ.
5. Принятие условий

Нажимая кнопку «Зарегистрироваться» и/или продолжая использование сервиса, пользователь подтверждает, что ознакомлен с настоящим пользовательским соглашением, полностью согласен с его условиями и обязуется их соблюдать.

Текст соглашения является упрощённой версией и может быть детализирован в официальной редакции документа.
