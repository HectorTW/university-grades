Правила работы сайта

Базовые правила использования сервиса «СтартТруд» для соискателей, работодателей и экспертов.

  1. Используйте реальные и актуальные данные в профиле.
  2. Не публикуйте оскорбительные, экстремистские или иные недопустимые материалы.
  3. Не передавайте свои учётные данные третьим лицам.
  4. Относитесь уважительно к преподавателям, работодателям и другим пользователям сервиса.
  1. Используйте сервис исключительно для поиска и оценки кандидатов.
  2. Не запрашивайте у студентов персональные данные сверх необходимых для трудоустройства.
  3. Не рассылайте массовые неперсонализированные сообщения (спам).
  4. Соблюдайте деловую этику и корректный тон общения.
  1. Выставляйте оценки объективно и обоснованно.
  2. Не используйте сервис для публичного унижения студентов.
  3. Не передавайте доступ к системе третьим лицам.
  4. Соблюдайте требования локальных актов и законодательства в части обработки персональных данных.

Нарушение правил может привести к ограничению доступа или блокировке аккаунта по решению администратора сервиса.
