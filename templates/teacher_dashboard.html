Инструкция по работе
Как выставлять оценки:
  1. Выберите группу студентов
  2. Укажите предмет
  3. Выставьте оценки по трем критериям:
    • Оценка - основная оценка (1-5)
    • Трудолюбие - прилежание (1-5)
    • Знания - уровень знаний (1-5)
  4. Добавьте комментарий (необязательно)
  5. Сохраните оценки
Советы:
  • Оценки выставляются по 5-балльной шкале
  • Система автоматически рассчитывает средний балл
  • Комментарии помогают студентам понять свои сильные и слабые стороны
  • Все оценки сохраняются с указанием даты и времени
